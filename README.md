# Game Dialogue Generator

A simple browser tool to create **visual novel style dialogue scenes** using your own character illustrations.

You can quickly generate mockups that look like a game screenshot — perfect for showcasing character art, creating visual novel concepts, or making fun dialogue scenes.

No installation required. Runs directly in your browser.

## Demo

Try it here:  
https://kuraymd.github.io/Game-Dialogue-Generator/

## Features

- Choose from **15 background scenes** (cafe, mansion, shrine, space, etc.)
- Place **up to 2 characters**
- Adjust **character size and position** with sliders
- Enter **character name, dialogue, and title**
- **Export the scene as an image**
- Upload **your own custom background**

## How to Use

1. Upload your character illustration
2. Choose a background
3. Adjust character size and position
4. Enter name and dialogue text
5. Export the final scene as an image

That's it!

## Who is this for?

This tool may be useful for:

- Artists selling **character illustrations / standing sprites**
- Game developers creating **visual novel mockups**
- Creators who want to preview **how their characters look in a game**
- Anyone who wants to make **fun dialogue scenes**
